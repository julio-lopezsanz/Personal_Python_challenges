# Personal Python Challenges

Repositorio personal donde recopilo retos prácticos en Python con el objetivo de mejorar mi forma de programar, escribir código más limpio, legible y pythonic, y reforzar buenas prácticas.

Este repositorio también funciona como material de repaso activo de temas fundamentales de Python.
Cada reto está pensado para practicar, cometer errores, corregirlos y entender por qué una solución es mejor que otra.

## Objetivos del repositorio

- Reforzar los fundamentos de Python mediante práctica constante.
- Mejorar la calidad del código (legibilidad, claridad y estilo).
- Aplicar buenas prácticas y patrones comunes.
- Detectar y corregir errores habituales.
- Preparar una base sólida antes de avanzar a Programación Orientada a Objetos (POO).

## Temas incluidos en el repaso

- Variables y tipos de datos
- Tipos mutables e inmutables
- Cadenas de texto y sus métodos
- Listas, tuplas y sus métodos
- Bucles for y while
- Funciones y alcance (scope)
- Uso de enumerate y zip
- Expresiones lambda
- Uso correcto de map y filter (cuándo usarlos y cuándo no)
- Diccionarios y patrones comunes:
- conteo
- acumulación
- agrupación
- filtrado
- transformación de datos
- Introducción a set
- Uso de la biblioteca estándar de Python
- Técnicas de depuración
- Manejo de excepciones
- Detección de errores comunes en Python

## Enfoque

Los retos priorizan:
- Soluciones claras antes que soluciones “inteligentes”
- Código entendible sobre código corto
- Aprender el por qué de cada decisión
